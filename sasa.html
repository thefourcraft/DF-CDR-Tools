CDR PRODUCTS

GateScanner

GateScanner is Sasa Software's flagship product, which is based on Content Disarm and Reconstruction (CDR) file sanitization technology
The GateScanner line of CDR-based network security solutions is designed to neutralize known and previously unknown threats by generating a functionally identical new copy of each incoming email and file that is devoid of any malicious content or links, transparently and in near real-time
GateScanner implements CDR technology to protect email, portable media uploads, cross-network file transfers, safe browser downloads, cloud-based file shares, and computing appliances
It supports hundreds of file types, including Microsoft Office files, PDF files, media files, archives, email formats, web and text files, medical imaging files, and other supported files
