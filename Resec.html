Resec

Resec offers a Zero Trust Prevention solution that safeguards organizations from file-based malware, including unknown threats. Their real-time data reconstruction and protection approach enhance endpoint security for enterprises.

CDR PRODUCTS

Resec's Platform

Resec's product is a cyber protection platform that eliminates all file-based malware threats without compromising usability
One of Resec's key products is its Content Disarm and Reconstruction (CDR) engine, which analyzes the content of all remaining files and rebuilds them into fully-functional and threat-free replicas
Resec's CDR engine supports over 50 file types
In addition to its CDR engine, Resec offers multiple solution modules that ensure protection from all common enterprise threat vectors

Supported File Types 

Based on the information available, Resec's CDR engine supports over 50 file types, including Emails, MS-Office, OpenOffice, Visio, PDF, HTML, Image, Archive, etc. Additional file types, including proprietary ones, can be added upon request.
